========= PARTIE 1 ==========

Présentation de la première ligne que l'on voit dans la console

cd : se déplacer
pwd : chemin du dossier actuel
ls : voir les fichiers et dossiers du dossier dans lequel on se situe (tout est fichier dans Linux)
	-a : voir les fichiers cachés (qui commencent par des points)
	-l : sous forme de liste avec des infos supplémentaires (notamment les droits)
cat / less / head / tail : voir le contenu du fichier
nano : éditer le contenu du fichier
	-m : utiliser la souris
touch : créer un fichier / mettre à jour la date de modification
mkdir : créer un dossier
rm : supprimer / renommer un fichier
	-R dossier
cp : copie
	-r dossier
mv : déplacer un fichier

grep recherche nomFichier : recherche un mot
	-n : numero de ligne
	-i : pas sensible à la casse
	-r : recherche dans le contenu du dossier

clear : nettoie la console
history : affiche la liste des commandes effectuées

whatis : description de la commande
man : manuel de la commande

ip addr : affiche les infos réseaux dont adresse IP

exit : sortir de votre session

sudo : droits d'admin

apt-get / yum install nomPaquet : installer un nouveau programme

========= PARTIE 3 ==========

sudo : mode root. 

su nomUser : changer d'utilisateur

sudo adduser nom (DEBIAN) 
sudo useradd nom (RED HAT) 

Un dossier personnel protégé dans /home est créé avec le nom de l'utilisateur.

sudo passwd nomUser : changer le mot de passe de l'utilisateur

Connaitre les utilisateurs : cat /etc/passwd

sudo deluser -r nom (DEBIAN)
sudo userdel -r nom (RED HAT)

sudo visudo : modifier les administrateurs du système

sudo addgroup nom (DEBIAN)
sudo groupadd nom (RED HAT)

sudo delgroup nom (DEBIAN)
sudo groupdel nom (RED HAT)

Savoir dans quel groupe est un utilisateur : groups nomUser 

Connaitre les groupes : cat /etc/group

sudo usermod nomUser : modifier un utilisateur
	-l : renomer l'utilisateur
	-g : changer le groupe de l'utilisateur
	-G : changer les groupes de l'utilisateur (séparés par des virgules)
	-a : ajoute l'utilisateur à 1 ou des gorupe(s)
	EXEMPLE : usermod -G amis, prof patrick

ls -l : voir les droits des fichiers

Explications : r, w et x

sudo chmod XXX nomFichier
	777 : tous les droits
	664 ou 660 : lecture / ecriture pour own et group,lecture seule ou rien pour les autres
	-R : sous dossiers et fichiers


sudo chown nomUser:nomGroup nomFichier -> changer le propriétaire d'un fichier
	-R

su nomUser 
exit
